# API de Notícias sobre Autismo

Esta é uma API desenvolvida em Java Spring Boot que fornece atualizações sobre notícias relacionadas ao autismo. A API é projetada para obter novos dados a cada 3 dias, garantindo que os usuários tenham acesso às informações mais recentes e relevantes.

## Funcionalidades
- **Atualizações periódicas:** A API verifica e atualiza as notícias a cada 3 dias.
- **Acesso a notícias:** Permite que os usuários consultem as notícias mais recentes sobre o autismo.
- **Formatos de resposta:** Suporta formatos JSON para fácil integração com outros sistemas.

## Tecnologias Utilizadas
- **Java Spring Boot:** Framework principal para desenvolvimento da API.
- **JPA/Hibernate:** Para gerenciamento de dados e persistência.
- **REST:** Para a construção da API e comunicação com os clientes.

## Instalação
1. Clone o repositório:
   ```
   git clone https://github.com/isabelaturina/api.noticia.git
   ```
2. Navegue até o diretório do projeto:
   ```
   cd api.noticia
   ```
3. Execute a aplicação:
   ```
   ./mvnw spring-boot:run
   ```
